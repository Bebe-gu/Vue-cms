# Vue商城项目

##环境依赖
node v12.4.0
vue v2.0 


##部署步骤
1. 添加系统环境包node_moudules
    先运行 npm install 安装依赖包

2. npm run dev  //开启服务

3. 浏览器项目运行

4. 预览
